## 🎯 О проекте

Комплексный анализ датасета бразильского маркетплейса **Olist** (99,441 заказов за 2016-2018 гг.). Проект демонстрирует навыки работы с SQL, Python, исследовательского анализа данных (EDA) и визуализации для получения бизнес-инсайтов.

### Ключевые бизнес-вопросы:
- ✅ Какие категории товаров наиболее прибыльны?
- ✅ Какова доля повторных покупок и как улучшить retention?
- ✅ Есть ли сезонность в продажах?
- ✅ Какие регионы требуют улучшения логистики?

## 💡 Ключевые инсайты

### 📈 Общая статистика
- **99,441 заказов** за 25 месяцев
- **96,096 уникальных клиентов** (некоторые сделали повторные покупки!)
- **97.02% успешно доставленных заказов** (96,478 из 99,441)
- **Средний чек:** 137.75 BRL (~₽2038,7 RUB)
- **Средний размер корзины:** 1.14 товара

### 🔄 Повторные покупки
- **Только 3.12% клиентов** вернулись за повторной покупкой (это очень низкий показатель по статистике, которую я нашел, на 2025 год - средний показатель удержания клиентов в магазинах электронной коммерции - 30-40%)
- **Основная проблема:** низкий retention rate
- **Рекомендация:** Внедрить программу лояльности

### 📦 Категории товаров
**Топ-3 по количеству заказов:**
1. 🛏️ Товары для дома (cama_mesa_banho) - 11,115 заказов 
2. 💄 Красота и здоровье (beleza_saude) - 9,670 заказов
3. ⚽ Спорт и досуг (esporte_lazer) - 8,641 заказов

**Топ-3 по выручке:** 
1. 💄 Красота и здоровье 
2. 💍 Часы и подарки 
3. 🛏️ Товары для дома

### 🚚 Логистика и доставка
- **Среднее время доставки:** 12.5 дней
- **Проблемные штаты:** Северные (22-29 дней доставки) и Северо-восточные штаты (18-24 дней доставки)

### 💳 Способы оплаты
1. **Credit card** - 73.9% (самый популярный)
2. **Boleto** - 19.0% (бразильский платежный метод)
3. **Voucher/Debit** - 7.1%

### 📅 Сезонность
1. **Главный пик продаж:** Ноябрь 2017 ("Black Friday") - 1.01 млн R$
2. **Второй пик продаж:** Апрель-Май 2018 - 1.00 млн R$ (стабильно высокие продажи)
3. **Сезонность:**
	- Рост в конце года (окт-ноя)
	- Спад в декабре после "Черной Пятницы"
	- Восстановление в начале года
4. **Аномалия:** Сентябрь (возможно, что-то не так с данными)

## 📝 Примеры SQL-запросов

### Анализ повторных покупок:
```sql
WITH customer_orders AS 
(
SELECT
    customer_unique_id AS Уникальный_клиент,
    COUNT(*) AS Количество_заказов
FROM customers
GROUP BY customer_unique_id
)
SELECT
    COUNT(CASE WHEN Количество_заказов > 1 THEN 1 END) AS Покупатели_с_повторной_покупкой,
    COUNT(*) AS Общее_количество_покупателей,
    ROUND(COUNT(CASE WHEN Количество_заказов > 1 THEN 1 END) * 100.0 / COUNT(*), 2) AS Процент_повторной_покупки
FROM customer_orders;
```

### Топ-3 товара по категориям:
```sql
WITH product_sales AS
(
SELECT
    p.product_category_name,
    oi.product_id,
    COUNT(*) AS sales_count,
    ROW_NUMBER() OVER(PARTITION BY p.product_category_name
                      ORDER BY COUNT(*) DESC) AS rank
FROM order_items oi JOIN products p USING (product_id)
GROUP BY p.product_category_name, oi.product_id
)
SELECT *
FROM product_sales
WHERE rank <= 3
ORDER BY product_category_name, rank;
```

## 🎯 Бизнес-рекомендации

### 1. Улучшение удержания клиентов (Retention)
**Проблема:** Только 3% клиентов возвращаются за повторной покупкой

**Рекомендации:**
- 🎁 Создать программу лояльности (скидки для постоянных клиентов)
- 📱 Push-уведомления о новых товарах в интересующих категориях

---

### 2. Оптимизация логистики
**Проблема:** Среднее время доставки 12.5 дней; северные регионы - 20+ дней

**Рекомендации:**
- 📍 Открыть дополнительные склады в северных штатах
- 🚚 Партнерство с локальными курьерскими службами

---

### 3. Фокус на высокомаржинальных категориях

**Рекомендации:**
- 💰 Программа рассрочки для дорогих товаров
- 📸 Улучшить качество фото и описаний
- 🎯 Таргетированная реклама на высокодоходные категории
